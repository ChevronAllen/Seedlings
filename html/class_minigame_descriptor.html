Detailed Description

A MinigameDescriptor is the object responsible for the data displayed on Screen when the minigame is selected. This means it will need:

  • Name
  • Description
  • Small Sprite (clickable)
  • Large Sprite (could just be a scaled version of the small sprite, but the larger version also could be more enhanced)
